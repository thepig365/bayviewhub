# Bayview Hub Host 使用手册

## 1. 这是什么

Bayview Hub 的 **Preliminary Passport** 是为作品建立的一份初步记录。它让一件作品先进入 Bayview 的记录体系，并生成一个可选择性分享的页面与二维码。

这不是认证书，不代表真伪鉴定、法律所有权确认、估值或销售承诺。

它的作用是：

- 先把作品记录下来
- 方便你选择性分享
- 让 Bayview 后续可在合适情况下介入 private viewing / curatorial consideration

---

## 2. 谁适合使用

你可以是：

- 收藏家
- 屋主 / Host
- 设计师 / Stylist
- 艺术家
- 其它与作品相关的持有人或登记人

只要你希望：

- 为一件作品建立更清晰的记录
- 生成一个可分享的作品页面
- 获得一个二维码用于墙边展示或邀请式分享
- 让 Bayview 在未来有机会考虑该作品用于线上策展或未来线下展陈计划

都适合使用。

---

## 3. 提交后你会得到什么

提交一件作品后，你通常会获得：

1. 一张 **Preliminary Passport 记录页**
2. 一个 **Passport ID**
3. 一个 **二维码（QR code）**
4. 一条可选择性分享的页面链接

该页面会显示：

- 作品主图
- 作品标题
- 艺术家姓名
- 材质 / 尺寸 / 年份（如有填写）
- 作品类型
- 备注说明
- Passport ID
- 状态（Preliminary）

不会公开显示：

- 你的住址
- 你的手机
- 你的邮箱
- 其它私人联系方式

---

## 4. 使用流程

### 第一步：进入注册页

打开：

- `/passport/register`

### 第二步：填写你的基本信息

包括：

- 姓名
- 邮箱
- 手机
- 区域 / suburb
- 你的角色（Your role）
- 可见性偏好

### 第三步：填写作品信息

包括：

- 作品标题
- 艺术家姓名
- 年份（可选）
- 材质
- 尺寸
- 持有状态
- 作品类型（原作 / 版画 / 数码输出 / 复制品）
- 作品说明 / 为什么它对你重要
- 是否希望未来 private viewing

### 第四步：上传作品主图

你至少需要上传一张**清晰的作品图片**。

建议：

- 光线均匀
- 画面尽量正
- 不要过度反光
- 尽量不要裁掉作品边缘

### 第五步：提交

提交成功后，系统会跳转到你的 **Preliminary Passport** 页面。

---

## 5. QR code 怎么用

二维码的作用是把别人带到这件作品的记录页。

你可以把它用于：

- 放在作品旁边
- 分享给受邀的人
- 私下发送给感兴趣的朋友或收藏家
- 作为通过 Bayview 进行 private viewing interest 的入口

不建议：

- 大范围公开传播
- 当作公开销售页面到处投放
- 把它理解为公开上架链接

它更适合 **选择性分享**，而不是公开广播。

---

## 6. “Curatorial Consideration” 是什么

提交成功后，你会看到一个板块：

**Would you like this work to be considered by Bayview?**

这表示你可以进一步表达：

- 仅保留为私人记录
- 希望 Bayview 考虑线上策展
- 希望 Bayview 考虑未来线下展陈计划
- 两者都考虑

这只是**表达兴趣**，不是自动入选。

Bayview 保留全部策展决定权。

表达兴趣并不代表：

- 一定会被选中
- 一定会上线展示
- 一定会在线下展出
- 一定会销售
- 一定会代理

---

## 7. 如果我是艺术家，也能用吗

可以。

现在注册页里的 **Your role** 已包含：

- Artist

所以艺术家本人也可以：

- 先为作品生成 Preliminary Passport
- 获取作品记录与二维码
- 再决定是否希望 Bayview 考虑该作品用于未来线上或线下策展

---

## 8. Bayview 会公开我的个人信息吗

不会。

Preliminary Passport 页面默认不会公开：

- 姓名
- 手机
- 邮箱
- 住址
- 直接联系信息

所有 viewing 相关的兴趣与后续联系，原则上都应通过 **Bayview 作为中介**。

---

## 9. 我提交后，Bayview 一定会联系我吗

不一定。

如果你只是生成 Preliminary Passport，并没有进一步表达策展兴趣，Bayview 不一定会主动跟进。

如果你表达了希望被考虑用于：

- online curation
- future in-person exhibition programming

Bayview 才可能在合适的时候联系你。

但是否跟进，仍取决于：

- 作品质量
- 与 Bayview 当前方向是否匹配
- 当期策展需求
- 展陈与资源安排

---

## 10. 我下一步最该做什么

如果你是 Host / 收藏家：

- 先完成 1–2 件代表性作品的登记
- 把 QR 放在作品旁边或保留好
- 只对合适的人选择性分享
- 如希望被考虑，勾选相应的 curatorial interest

如果你是 Artist：

- 先挑最能代表你的 1 件作品进入系统
- 确保主图清晰、信息完整
- 再表达希望 Bayview 考虑线上或未来线下策展

---

## 11. 常见问题

### Q1. Preliminary Passport 是正式认证吗？
不是。

### Q2. 它能证明作品真伪吗？
不能。

### Q3. 它能证明法律所有权吗？
不能。

### Q4. 它等于作品已被 Bayview 接受吗？
不等于。

### Q5. 我可以继续登记更多作品吗？
可以。提交成功页可继续选择 **Register Another Work**。

### Q6. QR code 是公开销售入口吗？
不是。它是一个选择性分享的记录入口。

---

## 12. 最后建议

把这套系统理解成：

- 先记录
- 再选择性分享
- 再决定是否希望进入 Bayview 的进一步考虑

它的第一价值不是“上架”，而是**为作品建立一个更清晰、更可被邀请式分享的入口**。

如果你愿意把作品进一步带入 Bayview 的语境，再表达 curatorial consideration 即可。
